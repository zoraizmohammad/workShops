### Description

This method lists all the available ML engines.

### Syntax

Here is the syntax:

```sql
db.ml_engines.find()
```

On execution, we get:

```sql
{
  NAME: 'lightwood',
  HANDLER: 'lightwood',
  CONNECTION_DATA: {
    key: [
      'password'
    ],
    value: [
      ''
    ]
  }
}
{
  NAME: 'openai',
  HANDLER: 'openai',
  CONNECTION_DATA: {
    key: [
      'password'
    ],
    value: [
      ''
    ]
  }
}
...
```
