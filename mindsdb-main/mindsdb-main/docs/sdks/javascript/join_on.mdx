## Description

The `runQuery()` function executes a query given as its argument directly in MindsDB.

## Syntax

Here is the syntax:

```
const query = `SELECT * FROM table_name t JOIN another_table a ON t…=a…`;
const queryResult = await MindsDB.SQL.runQuery(query);
```
